Welcome to my notes and summaries repository! 

This space was created to store and organize all my handwritten notes taken during classes and study sessions using a graphics tablet.

## 🎯 About the Files

To ensure the notes are accessible but also editable in the future, I save the documents in two main formats:

* 📄 **`.pdf`**: Exported versions of the notes. You can view them directly here on GitHub or in your browser, perfect for quick reading and review.
* 🖋️ **`.xopp` / `.xoj`**: Editable source files generated by the **Xournal++** application. If you want to edit, add notes, or modify the summaries, you will need to download these files.

## 🛠️ How to view and edit

1. **Read-Only:** Browse the folders and click on any `.pdf` file to view the notes directly on GitHub.
2. **To Edit:** Download the desired `.xopp` file.
   * Make sure you have [Xournal++](https://xournalpp.github.io/) installed on your machine (available for Windows, Linux, and macOS).
   * Open the file in the program and make your own annotations!

# Versão em português:

Bem-vindo ao meu repositório de anotações e resumos! 

Este espaço foi criado para armazenar e organizar todas as minhas anotações manuscritas feitas durante as aulas e sessões de estudo utilizando uma mesa digitalizadora. 

## 🎯 Sobre os Arquivos

Para garantir que as anotações sejam acessíveis, mas também editáveis no futuro, eu salvo os documentos em dois formatos principais:

* 📄 **`.pdf`**: Versões exportadas das anotações. Você pode visualizá-las diretamente aqui pelo GitHub ou no seu navegador, perfeitas para leitura rápida e revisão.
* 🖋️ **`.xopp` / `.xoj`**: Arquivos fonte editáveis gerados pelo aplicativo **Xournal++**. Se você quiser editar, adicionar notas ou modificar os resumos, precisará baixar esses arquivos.



## 🛠️ Como visualizar e editar

1. **Apenas Leitura:** Navegue pelas pastas e clique em qualquer arquivo `.pdf` para visualizar as anotações diretamente no GitHub.
2. **Para Editar:**  Baixe o arquivo `.xopp` desejado.
   * Certifique-se de ter o [Xournal++](https://xournalpp.github.io/) instalado na sua máquina (disponível para Windows, Linux e macOS).
   * Abra o arquivo no programa e faça suas próprias marcações!
